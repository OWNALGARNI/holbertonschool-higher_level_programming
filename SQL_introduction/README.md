The goal of this project is to learn the basics of **SQL** and **MySQL**, including database creation, table management, and data manipulation using SQL queries.

## Learning Objectives

By completing this project, you should be able to explain:

- What a database is
- What a relational database is
- What SQL stands for
- What MySQL is
- How to create and delete a database in MySQL
- The difference between DDL and DML
- How to CREATE and ALTER tables
- How to SELECT data from a table
- How to INSERT, UPDATE, and DELETE data
- What subqueries are
- How to use MySQL functions

## Environment

- OS: Ubuntu 22.04 LTS
- MySQL version: 8.0
- Allowed editors: `vi`, `vim`, `emacs`

## Requirements

- All SQL files are executed on **Ubuntu 22.04 LTS** using **MySQL 8.0**
- All SQL keywords must be written in **uppercase**
- All files must end with a new line
- Each SQL file must:
  - Start with a comment describing the task
  - Have a comment just before each SQL query
- A `README.md` file is mandatory
- File length will be tested using `wc`
