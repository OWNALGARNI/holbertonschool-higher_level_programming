# SQL - More queries

## Description
This project covers more advanced SQL concepts using **MySQL 8.0**:
- Creating MySQL users and managing privileges
- Using constraints (PRIMARY KEY, FOREIGN KEY, NOT NULL, UNIQUE)
- Querying data from multiple tables using JOINs
- Using subqueries and UNION
- Writing clean, readable SQL scripts following a style guide

## Learning Objectives
By the end of this project, I should be able to explain:
- How to create a new MySQL user and grant privileges
- What are PRIMARY KEY and FOREIGN KEY
- How to use NOT NULL and UNIQUE constraints
- How to retrieve data from multiple tables using JOIN
- What are subqueries
- What are JOIN and UNION

## Requirements
- OS: Ubuntu 20.04 LTS
- MySQL: 8.0 (tested on 8.0.25)
- Allowed editors: `vi`, `vim`, `emacs`
- All SQL files must end with a new line
- Each SQL file must start with a comment describing the task
- Each SQL query must have a comment just before it
- All SQL keywords must be in uppercase (SELECT, WHERE, JOIN, ...)
- A `README.md` file at the root of the project folder is mandatory
